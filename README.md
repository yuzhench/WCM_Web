# WCM: World-Cognition Model for Generalizable Human–Robot Interaction

Project page for **WCM**. Open `index.html`, or serve the folder and visit it in a browser.

## Contents
- `index.html` — the project page
- `media/img/` — teaser and CyberView figures
- `media/benchmark/` — XSquare benchmark demo videos
- `media/human/` — human–robot interaction demo videos
- `media/posters/` — video poster thumbnails

All videos are H.264 / yuv420p MP4 (web-compatible).
